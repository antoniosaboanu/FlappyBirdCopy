# FlappyBirdCopy - Unity 2D Development Project

A functional clone of the classic arcade game "Flappy Bird," developed to master the fundamentals of the **Unity Engine** and **C# scripting**.

## Project Overview
This was my first project in game development. The primary goal was to explore the Unity ecosystem and understand the core mechanics required to build a playable, physics-based 2D environment.

## Technical Features
* **Physics & Movement:** Implemented using Unity’s `Rigidbody2D` system to handle gravity and user-input-driven "flaps."
* **Collision Logic:** Utilized `BoxCollider2D` and triggers to detect pipe collisions and scoring zones.
* **Procedural Obstacles:** Developed a spawning system to manage the instantiation and recycling of obstacle pipes for infinite gameplay.
* **Game State Management:** Built a complete game loop including start menus, real-time score tracking, and game-over logic.

## Learning Outcomes
* **Unity Workflow:** Gained proficiency in using the Inspector, Prefabs, and Scene hierarchy.
* **C# Scripting:** Applied object-oriented programming principles to handle game logic and class communication.
* **Asset Integration:** Managed the pipeline for importing and utilizing 2D sprites and audio assets.
